Detailed Description

Core graphics engine functionalities.

The GraphicsEngine namespace provides rendering utilities, resource management, and graphical abstraction for different rendering techniques.

Typedef Documentation

◆ ICameraCallback

using GraphicsEngine::ICameraCallback = typedef Observable<glm::mat4, glm::mat4>::Callback

Alias for a callback function that takes two 4x4 matrices (view and projection).

This callback is typically used to handle updates to the camera's view and projection matrices. It is any function of the form void Fn(const glm::mat4& view, const glm::mat4& projection)
